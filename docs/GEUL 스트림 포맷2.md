GEUL Standard:
    0 = GEUL Standard 
    1 = GEUL Extension 

Kind (강제):
    0 = 노드 
    1 = 엣지

1비트(GEUL Standard) 0(GEUL Standard)이면,
    2비트(Kind)가 0(노드)이면,
        3비트가 0(개체/동사/컨텍스트)이면,
            4비트가 0(개체)이면,
                5-56비트:속성 약식 표현 비트
                57-64비트:중복방지용 아이디
            4비트가 1(동사/컨텍스트)이면,
                5비트가 0(동사)이면,
                    6-64비트:동사의 종류와 동사한정사 11종의 약식 표현
                        증거성(Evidentiality) 2비트 배정. 10:알수없음, 11:추론, 00:직접경험, 01:전언
                        서법(Mood) 2비트 배정. 10:알수없음, 11:가정, 00:서술, 01:명령
                        양태(Modality) 화자의 의지 정도. 2비트 배정. 10:알수없음, 11:의지 없음, 00:중립, 01:강한 의지
                        시제(Tense) 2비트 배정. 10:알수없음, 11:과거, 00:현재, 01:미래
                        상(Aspect) 비트마스크 3비트 배정. 1비트:진행, 2비트:완료, 3비트:결과중심
                        기간/시점(Period/Point) 특정 시점 또는 기간 정보. 1비트 배정. 0:Point, 1:Period
                        공손(Politeness) 2비트 배정. 10:알수없음, 11:반말, 00:중립, 01:존대
                        긍정/부정(Polarity) 2비트 배정. 10:알수없음, 11:부정, 00:중립, 01:긍정
                        의도성(Volitionality) 2비트 배정. 10:알수없음, 11:비의도적, 00:중립, 01:의도적
                        확신성(Confidence) 2비트 배정. 10:알수없음, 11:추측, 00:중립, 01:확신
                        반복성(Iterativity) 2비트 배정. 00:알수없음, 11:무한, 01:1회, 10:많음
                5비트가 1(컨텍스트)이면,
                    6-64비트:컨텍스트 약식 표현 비트
        3비트가 1(물리)이면,
            4비트 Unsigned:
                0 = signed 
                1 = unsigned
            5비트 Value Type:
                0 = 정수
                1 = 부동소수점
            6비트 확장여부:
                0 = 기본 물리량, 4워드 안에 헤더와 물리량 모두 표현
                    7-16비트:단위의 종류 1024종, 17-64비트:값
                1 = 확장 물리량
                    7-64비트:단위의 종류나 정보 설정, 단위의 종류에 따라 추가로 4워드 이상으로 값 표현
    2비트(Kind)가 1(엣지)이면,
        3-16비트:엣지의 종류 14bit 16,383종. 0x3FFF면 확장 워드.
1비트(GEUL Standard) 1(GEUL Extension)이면,
    3비트가 0(등록기관)이면,
        4비트가 0(엄격 심사)이면, 
            5-16비트:Issuer
        4비트가 1(최소 심사)이면,
            사업체이기만 하면 등록, 단 비인도적 비정상 사용시 회수
            5-32비트:Issuer
    3비트가 1이면,
        4비트가 0(자유)이면,
            완전 자유
        4비트가 1(미래)이면,
            GEUL NEXT GENERATION
