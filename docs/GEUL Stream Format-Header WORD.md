## 1. 📖 GEUL Header WORD

| 비트 범위 (Bits) | 필드 이름 | 비트 수 (Size) | 설명 |
| :--- | :--- | :--- | :--- |
| **1** | **V/O** (Version/Origin) | 1 bit | `0`: GEUL 표준, `1`: 사용자 정의 |
| **2** | **Ext** (Extension) | 1 bit | `0`: 표준, `1`: 확장 (미래 버전) |
| **3-5** | **T-High** (상위 종류) | 3 bits | 8개의 최상위 분류 |
| **6-16**| **T-Low** (하위 종류) | 11 bits | `T-High` 값에 따라 구조가 변하는 '컨텍스트 민감형' 제어 필드 (2048 슬롯) |

---

### 2. 🎛️ [3-5] T-High (상위 종류) (3비트)

파서가 즉시 분기하는 8개의 최상위 '서랍장'입니다.

* **`0` (0b000):** **동사** (TBD)
* **`1` (0b001):** **엣지** (이전 '문장 내 엣지' + '대명사 엣지' 통합)
* **`2` (0b010):** **메타** (생성 모델, Timestamp 등)
* **`3` (0b011):** **제어** (이전 '문장 간 엣지', 논리/인과관계)
* **`4` (0b100):** **명사** (개체 '선언' `의미 식별자`)
* **`5` (0b101):** **커밋** (버전관리)
* **`6` (0b110):** **물리량** (숫자, 시간, 단위 등)
* **`7` (0b111):** **종료**

---

### 3. 🧩 [6-16] T-Low (11비트) 컨텍스트별 분할 구조

이 설계의 핵심입니다. `T-Low` 11비트는 `T-High` 값에 따라 완전히 다른 '제어판'이 됩니다.

#### `T-High = 0` (동사)
* **`L-Index` (2비트):** '명시적 임시 인덱스'의 `L`값 (`0`:16, `1`:32, `2`:64, `3`:확장)
* **`Change` (1비트):** 변화 서술이 있는지
* **`State` (1비트):** 상태 서술이 있는지

#### `T-High = 1` (엣지)
* **`L-Source` (2비트):** Source 인덱스의 `L`값 (`0`:16, `1`:32, `2`:64, `3`:확장)
* **`L-Target` (2비트):** Target 인덱스의 `L`값
* **`Edge/Control Type` (7비트):** 엣지/제어의 종류 (128종)

#### `T-High = 2` (메타)
* **`Value Type` (2비트):** 수의 표현 방식 (`00`:FP, `01`:INT, `10`:UFP, `11`:UINT)
* **`L-Value` (2비트):** 값의 `L`값 (워드 길이)
* **`Unit Type` (7비트):** 메타 데이터의 종류 (128종)

#### `T-High = 3` (제어)
* **`L-Source` (2비트):** Source 인덱스의 `L`값 (`0`:16, `1`:32, `2`:64, `3`:확장)
* **`L-Target` (2비트):** Target 인덱스의 `L`값
* **`Edge/Control Type` (7비트):** 엣지/제어의 종류 (128종)

#### `T-High = 4` (명사)
* **`L-Index` (2비트):** '명시적 임시 인덱스'의 `L`값 (`0`:16, `1`:32, `2`:64, `3`:확장)
* **`L-Payload` (2비트):** '페이로드(ID)'의 `L`값
    * `0`: 대명사
    * `1`: 4 WORDs (64비트)
    * `2`: 8 WORDs (128비트)
    * `3`: 확장
* **`Feature Type` (7비트):** 명사의 최상위 특징 (128종) (e.g., 인간, 생물종)


#### `T-High = 6` (물리량)
* **`Value Type` (2비트):** 수의 표현 방식 (`00`:FP, `01`:INT, `10`:UFP, `11`:UINT)
* **`L-Value` (2비트):** 값의 `L`값 (워드 길이)
* **`Unit Type` (7비트):** 단위의 종류 (128종) (e.g., 미터, DFC)

#### `T-High = 7` (종료)
* `T-Low` 전체 11비트가 `111 1111 1111` (`0x7FF`)로 채워집니다.
